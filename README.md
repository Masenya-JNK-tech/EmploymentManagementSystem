# Employee Management System

## Overview

The Employee Management System is a Python application that helps manage employee records using a PostgreSQL database. It allows users to add, view, search, update, and delete employee information through both a terminal application and a Streamlit web dashboard.

---

## Features

- Add new employees
- View all employees
- Search employees
- Update employee information
- Delete employees
- PostgreSQL database integration
- Streamlit dashboard
- Employee statistics
- Data validation

---

## Technologies Used

- Python
- PostgreSQL
- Streamlit
- Pandas
- Matplotlib
- Psycopg2


## Installation

1. Clone the repository.

-bash
git clone https://github.com/Masenya-JNK-tech/EmployeeManagementSystem.git


2. Install the required packages.

-bash
pip install -r requirements.txt


3. Create a PostgreSQL database.

4. Update the database connection details in `database.py`.

5. Run the application.

Terminal version:

-bash
python main.py

Dashboard:

-bash
streamlit run dashboard.py
